HEADINGS <h1> - <h6>
These are used to define the headings in the page. h1 is the largest heading and h6 is the smallest subheading.
PARAGRAPHS <p>
These point out the paragraphs in a page.
BOLD <b>
This is used to make a character on the page bold.
ITALIC <i>
This is used to make a character on the page italicized.
SUPERSCRIPT <sup>
This is used to make a character on the page superscript. Smaller and higher than the baseline, an exponent.
SUBSCRIPT <sub>
This is used to make a character on the page subscript. Smaller and lower on the baseline, like in foot notes.
LINEBREAK <br />
This adds a new line within a paragraph.
HORIZONTAL RULE <hr />
This adds a new line between sections. This also adds an actual line where this element is.
STRONG <strong>
This tells the browser that whatever is in these tags is important and it will display as bold.
EMPHASIS <em>
When something needs to have a slight difference from the rest of the text.
BLOCK QUOTE <em>
This should be used for long quotes within text.
QUOTE <q>
This should be used for shorter.
ABBREVIATION <abbr>
This should be used to identify abbreviations or acronyms within text on a page.
DFN <dfn>
This should be used when describing a new term for the first time.
ADDRESS <q>
This should be used for shorter.
INSERT <ins>
When you want to show that new items have been inserted into the page you use this element.
DELETE <del>
When you want to show that new items have been removed from the page you use this element.
STRIKETHROUGH <s>
When you want to show an element on the page with a line through it you use this element.
ORDERED LIST <ol>
Creates a numbered list.
UNOREDED LIST <ul>
Creates a bulleted list.
LIST ITEM <li>
This defines an item within a list.
DEFINITION LIST <dl>
This creates a glossary.
DEFINITION TERM <dt>
Creates a new term in the glossary.
DEFINITION <dd>
The definition of the term created.
