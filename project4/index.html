My Favorite Camera

What's funny is I don't actually own this camera, I've only tested it a few times....but I love it! It's not the top of the line or anything, but for now it's my dream camera.

Nikon D7100

Nikon appears to have taken this challenge to heart with the D7100 looking, on paper, like a very significant upgrade.

Some of it's features:

  1. 24.1 effective megapixel CMOS, Nikon DX format image sensor, without an optical low-pass filter.
  2. Nikon EXPEED 3 image-processing engine.
  3. Advanced Multi-CAM 3500DX autofocus sensor module with 51 focus points.
  4. 3D Color Matrix Metering II 2,016-pixel RGB sensor.
